# Spotless Blinds Co.

Premium ultrasonic blind cleaning service for the San Francisco Bay Area.

## Website

Built with Next.js. Deployed on Vercel.

### Development

```bash
npm install
npm run dev
```

### Deployment

Automatically deploys to Vercel on push to `main`.
